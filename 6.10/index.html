Nadpis

Vytvořte základní webovou stránku (libovolný název) Stránku bude obsahovat klasické základní komponenty (nadpis, title, ikona atd.) Do stránky dodělejte smysluplně GRID a FLEXBOX (můžete udělat více stránek) Vytvořte soubor style.css, který připojíte k vašemu .html souboru a nastylujte stránku tak, aby byla uživatelsky přívětivá pomocí media queries ošetřete web tak, aby se při zmenšování velikosti stránky zmenšoval i grid a flexbox (zmenšuje se i počet sloupečků a případně řádků) pomocí media queries ošetřete web tak, aby se upravoval grid i flexbox v případě, že je obrazovku na šířku nebo na výšku. Výsledné řešení nahrajte na GitHub a udělejte Live Page -> výsledek si vyzkoušejte na mobilním telefonu i PC
